# LiDAR Obstacle Visualizer

A ROS 2 based system for real-time LiDAR obstacle detection and visualization.

## Features

- Real-time LiDAR scan visualization
- DBSCAN-based obstacle detection
- Left, Center, and Right zone monitoring
- Obstacle distance tracking
- Modern PyQt5 graphical interface

## Technologies

- Python
- ROS 2
- PyQt5
- NumPy
- SciPy
